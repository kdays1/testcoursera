HTML, CSS, and Javascript for Web Developers

  1. Responsive Site

    This page is meant to be responsive, if the width is reduced, its content is going to wrap into different columns and rows.

  2. Poly Restaurant

    This page simulates a Restaurant Page that is responsive, when the width is reduced, the menu turns into a drop-down menu and the logo disappears.

  3. Hello/Goodbye JS

    This page was the first approach to JS. In the console, there are some names and whenever the name begans with a "J" it should print "Googbye 'J'(name)", else, it should print "Hello (name)".

  4. David Chu's China Bistro (Random Specials)

    Using the template home page seen in the course, in the "Specials Menu" we had to choose a random menu and display it when the tile for specials was clicked.

Single Page Web Applications with AngularJS

  1. Lunch Checker

    This page receives an input from the user which should list things for lunch. If there are more than 3 objects, it's going to return a message saying is too much, if less than 3 objects it would say "Enjoy!".

  2. Shopping List Checker

    This page is a premade shopping list where you can click a button and buy the things in the list. Bought items will appear in the "items bought" list.

  3. Narrow Down Your Chinese Menu Choice

    This page let the user enter text and this input will be looked up in the description of the Chinese restaurant and retreive all the available options. Also, you can take out the options you don't like.
